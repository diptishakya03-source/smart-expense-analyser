# Smart Expense Analyser

A full-stack web application to track, categorize, and analyze personal expenses.

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

## Features
- Add expenses
- View expense list
- Expense categorization
- Basic expense analysis
