# Smart Expense Analyser

A full-stack web application to track, categorize, and analyze personal expenses with smart insights and alerts.

---

## Tech Stack
- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- Tools: VS Code, Git, GitHub

---

## Features
- Add expenses with date, category, amount, and notes
- View complete expense history
- Expense categorization
- Expense summary and visual analysis
- Smart spending alerts
- Expense health score
- Spending insights and suggestions
- Month-to-month expense comparison

## How to Run the Project Locally

1. Clone the repository
```bash
git clone https://github.com/diptishakya03-source/smart-expense-analyser.git

2. Navigate to the project directory
cd smart-expense-analyser

3. Create and activate virtual environment
python -m venv venv
source venv/bin/activate  
venv\Scripts\activate  

4. Install dependencies
pip install -r requirements.txt

5. Run the application
python app.py

6. Open in browser
http://127.0.0.1:5000
